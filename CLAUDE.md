# t-graph

A React component that renders 4 time-series in a single chart using 4 different rendering types: **area**, **spline**, **line**, **bar**. Built as a take-home test deliverable.

Stack: React 19, TypeScript, Vite, ApexCharts (via `react-apexcharts`).
